# Setup Instructions

- **JDK Version:** JDK 17 (or 21, whichever you are using)
- **IDE:** IntelliJ IDEA / VS Code
- **How to Run:**
1. Open the project in your IDE.
2. Navigate to `src/Main.java`.
3. Right-click and select 'Run Main.main()'.
4. You will see the "Welcome to LearnTrack System" message in the console.
